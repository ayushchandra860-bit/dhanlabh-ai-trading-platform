# DhanLabh Installation Guide

## Install The Desktop App

1. Open the `release-desktop` folder.
2. Double-click `DhanLabh-Setup-2.0.0.exe`.
3. Follow the installer.
4. Launch `DhanLabh` from the desktop shortcut or Start Menu.

## Portable Version

Use `DhanLabh-Portable-2.0.0.exe` if you do not want to install the app. Double-click it and the desktop assistant starts directly.

## First Launch

1. Open Olymp Trade.
2. Keep the main candlestick chart visible.
3. Start DhanLabh.

DhanLabh automatically finds the Olymp Trade window, locks onto the chart, and starts analyzing when the chart changes. If Olymp Trade is not open, it shows `Waiting for Olymp Trade`.

## Overlay

- Click `Overlay` to show the compact always-on-top view.
- Click `Move Overlay` when you want to reposition or resize it.
- Click `Lock overlay over chart` in Settings when you want it to become click-through so it does not block the chart.

## Local Data

Screenshots, settings, logs, and history are stored locally in the Windows app data folder for DhanLabh AI Chart Assistant.
